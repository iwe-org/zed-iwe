# Zed IWE extension

This Zed extension provides support for Markdwon documents graph navigation and tranformation using [IWE](https://iwe.md).

The main project repository is: https://github.com/iwe-org/iwe - Check the [README](https://github.com/iwe-org/iwe) for more information. 

Please feel free to ask questions in the [discussions](https://github.com/iwe-org/iwe/discussions) for the main repository.
